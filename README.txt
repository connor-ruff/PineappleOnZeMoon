# Pineapple Moon Base Solution

To execute the code:
    - Present working directory should be the directory where this README and all other submitted files are
    - Run the python file 'pineapple.py' with an optional command line argument specifying the file of Moon bases to read from
        Example:    python3 ./pineapple.py SampleMaps.txt
    - If you do not specify a CL argument, the default file will be SampleMaps.txt
    - The output of the code is to stdout
